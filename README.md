# IT3040 Assignment 1 - Option 1  
## Transliteration Accuracy Testing

This project contains the Playwright automation scripts used to test the Chat Sinhala transliteration function available at:

https://www.pixelssuite.com/chat-translator

The objective of this automation is to execute the prepared Singlish test cases and compare the expected Sinhala output with the actual Sinhala output generated by the application.

---

## Requirements

Before running the project, install:

- Python 3.11 or 3.12
- Google Chrome
- Playwright
- openpyxl

---

## Installation Steps

Open Command Prompt or VS Code terminal inside the project folder and run:

```bash
python -m pip install --upgrade pip
python -m pip install playwright openpyxl
python -m playwright install
